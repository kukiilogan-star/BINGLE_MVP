# NP1 얼음요정 (빙글이들) 물리엔진 및 온도 시스템 기능 정의서

## 1. 핵심 컨셉: 온도(Temperature) 기반 상태 변화
- **기존 기획:** 우측 상단에 Happiness(행복도) 및 Level 표시
- **변경 기획:** 우측 상단에 **온도바(Temperature Bar)** 배치
- **목적:** 사용자가 게임과 상호작용하지 않거나 방치할 경우, 냉장고의 온도가 올라가며 얼음 요정들이 서서히 녹아내리는(Melting) 시각적 피드백 제공

## 2. 상태 변화 및 에펙 효과 (After Effects Style)
- 온도가 일정 수준 이상 올라가면 얼음이 녹기 시작합니다.
- **스프라이트 시트 애니메이션:** 단순한 CSS 형태 변화가 아닌, 4프레임 이상의 **스프라이트 시트(Sprite Sheet)**를 활용하여 에프터이펙트(After Effects) 수준의 디테일하고 생동감 넘치는 녹아내림(Melting) 애니메이션을 구현합니다.
- **물리 엔진 연동 (Matter.js):** 
  - 얼음 요정들은 서로 충돌하거나 선반 바닥과 상호작용합니다.
  - 녹아내릴수록 물리 엔진 상의 충돌 판정 박스(Hitbox)가 작아지거나, 액체처럼 부드러운 소프트바디(Soft Body) 물리 법칙이 적용됩니다.

## 3. 사용자 상호작용 (온도 유지)
- 사용자가 냉장고 하단 버튼(`간식 주기`, `놀아 주기`, `집 꾸미기`)을 클릭하거나 얼음 요정을 드래그하며 놀아줄 때마다 온도가 다시 내려가며 본래의 단단한 얼음 형태를 회복합니다.
- 탭(Tap)할 때 나오는 말풍선 애니메이션 역시 물리 엔진의 반발력(Restitution)과 연동되어 통통 튀는 효과를 줍니다.
